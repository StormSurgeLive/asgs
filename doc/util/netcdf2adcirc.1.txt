netcdf2adcirc(1)
================
:doctype: manpage

NAME
----
netcdf2adcirc.f90 - Converts ADCIRC netcdf output files to ADCIRC ascii. 

SYNOPSIS
--------
*netcdf2adcirc.x* [--sparse] --datafile 'ADCIRC_FILE_NAME'


DESCRIPTION
-----------
The +netcdf2adcirc.f90(1)+ program converts ADCIRC netcdf output files to 
ADCIRC ascii output files. The *--datafile* argument is required.
The +netcdf2adcirc+ program is 
used in cases where netcdf output files were generated by ADCIRC, but 
legacy post processing requires ascii format. 

OPTIONS
-------

*--sparse*:: 
   Write sparse ascii files; that is, only write data values at wet 
   nodes. 

*--datafile* 'ADCIRC_FILE_NAME'::
    Use the netcdf formatted ADCIRC output file named 'ADCIRC_FILE_NAME' when
    writing data to the ascii file. The *netcdf2adcirc* program supports
    the following ADCIRC output files: 'fort.45.nc', 'fort.63.nc', 
    'fort.64.nc', 'fort.73.nc', 'fort.74.nc', 'maxele.63.nc', 'maxwvel.63.nc',
    'swan_DIR.63.nc', 'swan_HS.63.nc', 'swan_TMM10.63.nc', 
    and 'swan_TPS.63.nc'.
